身为一名程序员，一直想动手搭建一个独立博客，记录自己的技术成长之路。第三方博客使用起来有点不自在，始终得不到真正的自由，也失去了自主的定制性和创造性。幸好，在编程界，有限制的地方，就会有解决限制的伟大而优秀的程序员们，他们被称之为黑客（Hacker），一群不服从教条、崇尚自由、喜欢挑战难题的技术狂人。Hexo 和 GitHub ，正是在这样的背景下诞生的优秀开源产物，有了它们，我的 Hacking 之路将会变得更加有趣。

环境准备（window）

安装Node.js

到 Node.js 官网下载相应平台的最新版本，然后一路安装即可。

安装Git

Git的客户端很多，本人使用的是 msysgit，下载后设置下环境变量即可。

安装编辑器（可选）

Sublime Text 2，支持各种编程语言和文件格式，Markdown自然也不会例外。除了这个之外，还有其他很多不错的编辑器选择，萝卜青菜，各有所爱了。这里提供个绿色版本下载 Sublime Text 2.0.2

其他平台可以参考官方说明：hexo installation

GitHub 配置

  1. 注册一个 GitHub账号
  2. 建立一个与GitHub用户名相同的 repo，repo 的名称必须为：your_user_name.github.io
  3. 添加本地电脑SSH公钥到GitHub账户：Account settings -> SSH Keys -> Add SSH Key

前两步完成简单，着重介绍第3步：

执行成功后会在用户目录下生成一个.ssh目录，里面有id_rsa和id_rsa.pub两个文件，这两个就是SSH Key的秘钥对，id_rsa是私钥，不能泄露出去，id_rsa.pub是公钥，可以放心地告诉任何人。
